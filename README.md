# AIOps Platform - Hands-On Engineering

🚀 **Гибридная AIOps-платформа с AI/ML детекцией и автоматическим восстановлением**

## Архитектура

Проект представляет собой подробную разработку по созданию и развертыванию гибридного AIOps-решения. AIOps (Artificial Intelligence for IT Operations) – это применение ИИ и машинного обучения к ИТ-операциям для автоматизации и улучшения задач мониторинга, управления инцидентами и других операционных процессов.

### Технологический стек

| Компонент | Инструмент | Обоснование |
|-----------|------------|-------------|
| Оркестратор | Rundeck | Прост, расширяем, поддерживает скрипты и SSH out-of-the-box |
| AI/ML ядро | Go + GoLearn | Производительность + контроль (ML-наборы: Golearn, Gorgonia) |
| Контейнеризация | Kubernetes (K3s) | Лёгкий в разворачивании, идеален для локального девелопа |
| Мониторинг | Prometheus | Де-факто стандарт |
| Логирование | Loki + Promtail | Нативно интегрируется с Grafana |
| Расширенный лог | Elasticsearch + Fluentd | Масштабируемо, мощный анализ |
| Визуализация | Grafana | Стандартная панель для DevOps |
| Секреты | Vault | Безопасность автоматизации |
| IaC | Terraform | Полностью описываемая архитектура |
| Provisioning | Ansible | Декларативное управление вне Kubernetes |

### Гибридная модель

Наш подход сочетает:
- **Интеллектуальный анализ данных** (AI/ML-модуль на Go)
- **Надежный оркестратор** (Rundeck) для автоматического выполнения сценариев исправления (auto-remediation)

Такая гибридная модель позволяет построить самовосстанавливающуюся (self-healing) архитектуру, где система сама обнаруживает проблемы, принимает решения и выполняет действия по восстановлению, минимизируя время простоя и ручное вмешательство.

## Быстрый старт

### Предварительные требования

- K3s (установлен с ZFS storage: `/k3s-storage-pool/data`)
- Go 1.21+
- Docker
- kubectl
- Terraform
- Ansible

### Развертывание

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   cd aiops-dev
   ```

2. **Инициализация Go модуля**
   ```bash
   go mod init aiops-platform
   ```

3. **Развертывание инфраструктуры**
   ```bash
   make deploy-infra
   ```

4. **Запуск AI/ML компонентов**
   ```bash
   make start-ml
   ```
