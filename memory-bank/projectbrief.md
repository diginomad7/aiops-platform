# AIOps Platform - Project Brief

## Цель проекта
Разработать и внедрить AIOps-платформу с гибридной моделью AI/ML (на Go) + оркестрация (на Rundeck), развернутую в Kubernetes с полной обсервабилити, логированием, auto-remediation и визуализацией.

## Описание
AIOps (Artificial Intelligence for IT Operations) – это применение ИИ и машинного обучения к ИТ-операциям для автоматизации и улучшения задач мониторинга, управления инцидентами и других операционных процессов.

## Архитектура решения
Гибридная модель, сочетающая:
- **Интеллектуальный анализ данных** (AI/ML-модуль на Go)
- **Надежный оркестратор** (Rundeck) для автоматического выполнения сценариев исправления

## Технологический стек
- **AI/ML ядро**: Go + GoLearn
- **Контейнеризация**: Kubernetes (K3s) 
- **Мониторинг**: Prometheus
- **Логирование**: Loki + Promtail
- **Визуализация**: Grafana
- **IaC**: Terraform
- **Provisioning**: Ansible

## Модули разработки
1. **Модуль 1**: Основы и Подготовка Инфраструктуры (2 недели)
2. **Модуль 2**: Оркестрация и Автоматизация (2 недели)  
3. **Модуль 3**: Расширенные Возможности (2 недели)
4. **Модуль 4**: Интеграции и Визуализация (2 недели)

## Результат
Самовосстанавливающаяся (self-healing) архитектура, где система сама обнаруживает проблемы, принимает решения и выполняет действия по восстановлению.
