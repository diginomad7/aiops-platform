# План обучения AIOps
Архитектура курса: «AIOps Hands-On Engineering»
💡 Цель: разработать и внедрить AIOps-платформу с гибридной моделью AI/ML (на Go) + оркестрация (на Rundeck), все развёрнуто в Kubernetes с полной обсервабилити, логированием, auto-remediation и визуализацией.
Проект представляет собой подробную разработку по созданию и развертыванию гибридного AIOps-решения. AIOps (Artificial Intelligence for IT Operations) – это применение ИИ и машинного обучения к ИТ-операциям для автоматизации и улучшения задач мониторинга, управления инцидентами и других операционных процессов. Наш подход сочетает интеллектуальный анализ данных (AI/ML-модуль) с надежным оркестратором (Rundeck) для автоматического выполнения сценариев исправления (auto-remediation). Такая гибридная модель позволяет построить самовосстанавливающуюся (self-healing) архитектуру, где система сама обнаруживает проблемы, принимает решения и выполняет действия по восстановлению, минимизируя время простоя и ручное вмешательство.
